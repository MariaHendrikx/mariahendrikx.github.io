About the Challenge

Hello! I'm Maria Hendrikx, a passionate developer that started a new challenge: building 100 different startups. This challenge is about exploring what drives people to buy something and what sticks. I'll be documenting the entire journey.

Two years ago, the original plan was to build many startups, see whatever takes off and go for that one. However, I had done this already in 2019 on a small scale and after 2 weeks it felt a bit robotic. Also, the semester started a bit later again, so I wouldn't be able to sustain the focus.

The Golden Rule

If you need a tutorial, I failed. The startups must be incredibly easy to use.

Challenge Rules

  • The project must be easy to use: If you need a tutorial, I failed
  • It should be monetizable (not necessarily from day 1, but along the way)
  • I'm allowed to work on builds together with other people
  • Any tools allowed, any type of build allowed (webpage, mobile app, or even hardware)
  • Can be a teeny tiny mini-project or something super ambitious
  • Must make a blog/report/reflection about each project
  • Need to answer the same set of questions for each app
